# SCalka – Native ELF Calculator for Siemens Mobile Phones

SCalka is an ELF application that replaces the built‑in Java calculator on Siemens mobile phones. It provides a fast, native interface with all standard arithmetic operations and optional advanced features.

## Building
The SDK must be located at `../sdk` relative to the project root.
```bash
mkdir build && cd build
cmake ..
make
```
